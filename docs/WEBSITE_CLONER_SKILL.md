# Website Cloner Skill Notes

This repo includes `skills/website-cloner/SKILL.md`, adapted from the MIT-licensed `JCodesMore/ai-website-cloner-template` workflow.

The skill is used as a disciplined research-and-rebuild process:

- inspect the target site with browser automation;
- extract observable design tokens, behaviors, topology, and implementation constraints;
- write local research artifacts;
- translate the design language into original Cup Signal code and assets;
- verify with build, browser screenshots, and live-page checks.

For third-party inspiration sites such as Active Theory, this project does not copy proprietary bundles, brand assets, fonts, copy, images, or videos. It translates observable interaction principles into original implementation: black cinematic stage, pointer-reactive canvas field, sparse oversized typography, kinetic rails, and hard-edged studio panels.
